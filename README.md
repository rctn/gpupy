gpupy
=====

Wrapper for cuBlas\NumbaPro functions that provides drop in support for NumPy functions on GPU

### Use
Instantiate a Gpupy object 
```python 
from gpupy import Gpupy
gp = Gpupy()
```
Functions:

gp.dot - np.dot drop-in
